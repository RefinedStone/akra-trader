Admin Control Room UI Research

2026-05-04

Lazyweb MCP and screenshot browse tooling are not exposed in this environment, so no Lazyweb database screenshots could be downloaded. The current implementation was inspected from the React shell, global CSS, and route composition.

Direction

  • Favor command-center information density over marketing-style hero composition.
  • Keep persistent workspace navigation with a stronger active state.
  • Put status and API context near the top for fast operator orientation.
  • Use calmer tokens with purposeful accent colors for operational states.
  • Improve form, table, card, and button consistency globally.
